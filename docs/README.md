# kuDOS

The minimum requirements are as follows:
* 2 MiB of RAM
* i386SX
* PC/AT compatible
* 20 MiB hard drive (note binary measurments)

The recommended hardware:
* 4 MiB of RAM
* i386DX
* PC/AT or PS2
* 30 MB

The maximum amount of ram is 1 GB.

# Uninstall

The installation can be easily reversed. Edit config.sys and remove DEVICE=KUDOS\BOOT386.SYS. After restarting the PC, MS-DOS will run.  It is safe to delete the entire KUDOS directory.
